# UK AI Forum: Using Inspect for AI Safety Research

Demo materials for the "Using Inspect for AI Safety Research" workshop at UK AI Forum

## Structure

The demo walks through:
- how to build a basic evaluation in Inspect
- how to analyse results using the Inspect Log Viewer and Weights & Biases
- how to run a more complex benchmark from Inspect Evals

You can get started by following the instructions in [TUTORIAL.md](./TUTORIAL.md)

## Colab

You can also follow the demo by making a copy of [this Colab notebook](https://colab.research.google.com/drive/1hngkyhB6c2fNNLTxjz0j6pQXISKjIbrw?usp=sharing) and working through it.
